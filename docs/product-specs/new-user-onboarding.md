# New User Onboarding

> 이 문서는 제품 스펙 작성의 템플릿 예시다. 실제 프로젝트에 맞게 수정하여 사용한다.

## 개요

신규 사용자가 서비스에 가입한 후 핵심 가치를 경험하기까지의 플로우를 정의한다.

## 배경 & 동기

- 신규 사용자의 첫 경험이 리텐션을 결정한다
- 가입 후 핵심 액션까지의 시간을 최소화해야 한다

## 사용자 스토리

- 사용자로서, 가입 후 30초 이내에 서비스의 핵심 기능을 체험하고 싶다
- 사용자로서, 복잡한 설정 없이 바로 시작하고 싶다

## 기능 요구사항

- [ ] 소셜 로그인 (Google, Apple)
- [ ] 최소한의 프로필 설정 (이름, 프로필 사진)
- [ ] 온보딩 튜토리얼 (3단계 이내)
- [ ] 스킵 가능한 설정 단계

## UI/UX 플로우

```
가입 → 프로필 설정 → 튜토리얼 (스킵 가능) → 메인 화면
```

## 엣지 케이스

- 소셜 로그인 실패 시 이메일 폴백
- 네트워크 끊김 시 오프라인 안내
- 이미 가입된 이메일로 재가입 시도

## 성공 지표

- 가입 완료율: > 80%
- 온보딩 완료율: > 60%
- 첫 핵심 액션까지 시간: < 2분
