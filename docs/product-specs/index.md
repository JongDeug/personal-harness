# Product Specs Index

제품 스펙 문서 목록. 새로운 기능이나 사용자 플로우를 정의할 때 작성한다.

## 작성 가이드

제품 스펙은 다음 구조를 따른다:

1. **개요** — 한 문장으로 요약
2. **배경 & 동기** — 왜 이 기능이 필요한가
3. **사용자 스토리** — 누가, 무엇을, 왜
4. **기능 요구사항** — 구체적인 기능 목록
5. **UI/UX 플로우** — 화면 흐름 (스크린샷 또는 와이어프레임)
6. **엣지 케이스** — 예외 상황 처리
7. **성공 지표** — 어떻게 성공을 측정할 것인가
