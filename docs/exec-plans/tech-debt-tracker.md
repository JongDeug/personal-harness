# Tech Debt Tracker

기술 부채를 인식하고 추적한다. 부채를 만들 때 여기에 기록하고, 해결하면 제거한다.

## 분류 기준

- **P0 (Critical)** — 서비스 안정성 또는 보안에 직접 영향. 즉시 해결
- **P1 (High)** — 개발 속도를 눈에 띄게 저하. 다음 스프린트에 해결
- **P2 (Medium)** — 불편하지만 우회 가능. 여유 있을 때 해결
- **P3 (Low)** — 개선하면 좋지만 급하지 않음
