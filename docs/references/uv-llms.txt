uv Reference for LLMs
=====================

uv는 Rust로 작성된 초고속 Python 패키지 매니저다.
pip, pip-tools, virtualenv를 대체한다.

## 기본 사용법

```bash
# 프로젝트 초기화
uv init

# 의존성 추가
uv add <package>
uv add --dev <package>

# 의존성 설치 (lockfile 기반)
uv sync

# 스크립트 실행
uv run python main.py
uv run pytest

# 가상환경 생성
uv venv
```

## pyproject.toml 연동

uv는 pyproject.toml의 [project.dependencies]를 사용한다.
uv.lock 파일로 재현 가능한 빌드를 보장한다.

## pip 호환 명령

```bash
uv pip install <package>
uv pip compile requirements.in -o requirements.txt
uv pip sync requirements.txt
```
