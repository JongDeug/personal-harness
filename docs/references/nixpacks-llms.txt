Nixpacks Reference for LLMs
============================

Nixpacks는 소스 코드로부터 OCI 이미지를 자동 빌드하는 도구다.
Railway 등에서 기본 빌드 시스템으로 사용된다.

## 기본 동작

- 소스 코드의 언어/프레임워크를 자동 감지
- Nix 패키지를 사용하여 빌드 환경 구성
- Dockerfile 없이 컨테이너 이미지 생성

## 주요 설정 (nixpacks.toml)

```toml
[phases.setup]
nixPkgs = ["...", "..."]

[phases.build]
cmds = ["..."]

[start]
cmd = "..."
```

## 언어별 기본 감지

- Node.js: package.json 감지 → npm/yarn/pnpm install → npm start
- Python: requirements.txt 또는 pyproject.toml 감지
- Go: go.mod 감지
